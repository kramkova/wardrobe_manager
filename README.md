# Wardrobe Manager
Мини-сервис на базе Python и SQL для хранения и обработки данных о ваших предметах гардероба и их использовании.

## Что есть в репозитории?
src/ - Основная директория проекта. 

tests/ - Тесты для разработчика

.pre-commit-config.yaml - Настройки прекоммита (Ruff, unittests) для разработчика

poetry.lock - Настройки виртуальной среды poetry

pyproject.toml - Описание проекта, зависимости

README.md - Настоящая инструкция

## Как это работает?
Клонируйте репозиторий, установите зависимости, запустите src/main.py. Взаимодействуйте с помощником через терминал: введите пользовательское имя, добавьте информацию о своих вещах и образах из них, получите рекомендации новых образов на основе истории сочетаний.

После закрытия программы вашу базу данных можно будет найти в папке wardrobe_manager\src\databases. При новом запуске программы помощник сможет снова подключиться к вашей базе, когда вы введёте имя.

## Что дальше?
В следующих версиях планируется:
- настроить обработку крайних случаев
- сделать ввод данных более гибким
- учитывать цвет при вычислении сочетаемости
- выводить любимые цветовые сочетания
- добавить поиск вещей и образов по фильтрам
- рекомендовать образы по параметрам (например, по случаю или сезону)
