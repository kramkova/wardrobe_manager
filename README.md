## Как это работает?

(Непонятно, как настроить окружение, как вводить данные, что будет в результате.)

Клонируйте репозиторий, установите зависимости, запустите src/main.py. Взаимодействуйте с помощником через терминал: введите пользовательское имя, добавьте информацию о своих вещах и образах из них, получите рекомендации новых образов на основе истории сочетаний.

После закрытия программы вашу базу данных можно будет найти в папке wardrobe_manager\src\databases. При новом запуске программы помощник сможет снова подключиться к вашей базе, когда вы введёте имя.
